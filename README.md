## Sobre mim

Sou Pedro Sales, estudante de Engenharia de Software no IFPB. Gosto de transformar ideias em aplicações web, conectando front-end, back-end, banco de dados e automações para criar experiências úteis e organizadas.

No momento, venho aprofundando minha base em desenvolvimento full stack, boas práticas de projeto, deploy e integrações entre ferramentas.
